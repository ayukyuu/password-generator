A clean, minimal password generator built with vanilla HTML, CSS and JavaScript — no frameworks, no dependencies, just handwritten code.

## ✨ Features

- Generates two random 16-character passwords at once
- Uses a wide character set: lowercase, uppercase, numbers and special characters
- Sleek dark UI with a custom Inter typeface via Google Fonts

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

## 🚀 Getting Started

No setup needed. Just clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/ayukyuu/password-generator.git
cd password-generator
open index.html
```
